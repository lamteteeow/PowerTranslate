# Privacy Policy

Last updated: 2026-04-07

PowerTranslate Extension is designed to minimize data collection and keep user control local.

## What Data Is Sent

- Text you choose to translate is sent to DeepL over HTTPS for translation.

## What Data Is Stored Locally

- DeepL API key: stored locally in encrypted form using Windows Data Protection API.
- Source and target language preferences: stored locally as non-sensitive settings.

## What We Do Not Collect

- No telemetry.
- No analytics.
- No usage profiling.
- No advertising identifiers.

## Third-Party Services

- DeepL API is used only to process translation requests.
- DeepL handling of request data is governed by DeepL terms and privacy policy.

## Data Control

You can remove local extension data at any time by deleting the extension cache files in the app local package storage location.

## Compliance Notes

- No intentional collection of personal data by this extension.
- The extension behavior is compatible with GDPR/CCPA principles of data minimization and user control.

## Contact

For privacy questions, open an issue in this repository.
